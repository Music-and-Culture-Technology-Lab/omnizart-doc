Command Line Interface¶

This page describes the basic concept and usage of omnizart command.

The root entry is omnizart, and followed by several sub-commands. To see a list of available sub-commands, type omnizart --help.

More detailed descriptions and the usage of each sub-commands can be found from their own page.

Transcribe¶

Collections of transcription commands.

Transcribes the audio in different aspects, such as instrument, drum, and vocal. Specify all to the first argument to transcribe all different aspects. The output will be a MIDI file with the same name as the input audio, except the extension will be replaced with ‘.mid’.

 Available functions are:

  • music - trancribes instrument notes

  • drum - transcribes drum percussions

  • vocal - transcribes vocal melodies

  • beat - MIDI-domain beat tracking

  • chord - MIDI-domain chord transcription

Input for the former three should be wav file, and the last two should be MIDI file.

Example Usage

$ omnizart transcribe music \ 
    example.wav \ 
    --model-path path/to/model \ 
    --output example.mid
